## 长说明（放 GitHub Pin / 个人简介链接页）

**标题：** 使用 Grok Build 在世界杯躺着赚钱的工具来啦

**简介：**

WC Arb Agent 是一款面向 2026 世界杯的开源套利工具，已打包为 **Grok Build 插件**。安装后，Grok 可自动扫描 Stake、Cloudbet 等平台的赔率差异，识别 surebet 与亚盘中间盘（middle），并按你的 USDT 本金计算每边下注金额。完成一次性平台登录后，支持 autopilot 自动下单。

**适合谁：**
- 用 USDT 在 crypto 体育盘下注的用户
- 已在使用 Grok Build 的开发者
- 想了解跨平台套利逻辑的人

**安装：**
```bash
grok plugin install AgenBot11/wc-arb-agent --trust
```

**Grok 常用指令：**
| 你说 | Grok 调用 |
|------|-----------|
| 初始化 | `wc_arb_onboard` |
| 扫描套利 | `wc_arb_scan` |
| 登录平台 | `wc_arb_login` |
| 开启自动下注 | `wc_arb_autopilot_enable` |
| 执行套利 | `wc_arb_autopilot` |

**重要说明：**
- 工具免费开源；The-Odds-API 等需 **用户自己注册 Key**（BYOK）
- 自动下注需显式开启 `autopilot --enable`
- 套利在实际操作中仍可能因赔率变动、延迟、限额等产生风险
- 仅供学习研究，请遵守当地法律与平台服务条款

**链接：** https://github.com/AgenBot11/wc-arb-agent
